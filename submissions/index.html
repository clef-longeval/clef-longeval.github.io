Submissions

Submissions will be done using through CodaLab compition, you can access COMING SOON . Each team will need to participant in the competition and submit their runs through it.

All the participants need to provide results for each submitted system on certain snapshot test sets which we will inform about soon. Participants can submit up to 10 systems and are required to provide descriptions on each of the systems trhough a dedicated form.

The individual runs need to be submitted in the TREC format. For each query in each run, it is allowed to return up to 1000 documents.

Details on specific submission file format will be provided soon.
